## 🏗️ Project Architecture
The codebase follows a modular "Page-Component" separation of concerns:

```text
src/
├── assets/images    # Backgroundless "Coding Man" parallax assets
├── components/      # Reusable UI (FloatingMan.jsx, ProjectCard.jsx)
├── pages/           # High-level page components (Home, About, Skills)
├── hooks/           # Framer Motion scroll logic
└── styles/          # Tailwind v4 Unified CSS
